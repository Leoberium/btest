**Install**:  
`git clone https://github.com/Leoberium/btest.git`

**Test**:  
While in `btest` directory run `pytest`

**Run**:
- To run server: `python3 server.py`  
- To run client with some queries: `python3 client.py`


Python modules required:
- `flask` (for server)
- `requests` (for client)
- `pytest` (for testing)
